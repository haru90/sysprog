# myFTP

## Overview
簡易的なFTPサーバ／クライアントプログラムです．  
現在実装中です．


## Description
- ファイルの送受信
- クライアント側でpwd・cd・lsに相当するコマンドを実行可能（サーバ／クライアント両方を対象に実行可能）


## Requirements
- gcc


## Installation
`$ make`


## Usage
### Server (myftpd)
`$ ./chat [current directory (option)]`

### Client (myftpc)
`$ ./myftpc [server hostname]`
